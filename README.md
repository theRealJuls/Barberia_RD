# Barberia_RD
Una pagina web responsive para barberias que permite agendar citas y tener un servicios mas placentero para el cliente al momento de recortarse el pelo.

## Modulo Brandon - Servicios y pagos

Rama: `feature/servicios-pagos`

Alcance:

- CRUD de servicios y precios.
- Registro de pagos en efectivo.
- Dashboard administrativo con metricas de servicios e ingresos.
- Base de datos en Supabase PostgreSQL.
- Frontend en JavaScript y React.

Archivos principales:

- `front-end/index.html`
- `front-end/package.json`
- `front-end/.env.example`
- `front-end/src/App.jsx`
- `front-end/src/main.jsx`
- `front-end/src/styles.css`
- `Data-base/schema.sql`
